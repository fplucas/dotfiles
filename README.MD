# fplucas' dotfiles

These are my dotfiles. Used on an Archlinux machine.

### Programs which have configs in this repository

- neovim (text editor)
- ranger (file manager)
- termite (terminal)
- zsh (shell)

## Installing

To install, just run:

```
git clone https://github.com/fplucas/dotfiles.git
cd dotfiles
./install.sh
```
