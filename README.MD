# fplucas's dotfiles

These are my dotfiles. Used in an Archlinux machine.

### Programs which have configs in this repository
- dunst (notification daemon)
- i3 (i3-gaps)
- nitrogen (wallpaper setter)
- neovim (text editor)
- polybar (status bar)
- ranger (file manager)
- termite (terminal)
- transmission (torrent client)
- zsh (shell)

## Installing

To install, just run:

```
git clone https://github.com/fplucas/dotfiles.git
cd dotfiles
./install.sh
```
