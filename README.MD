# fplucas' dotfiles

These are the dotfiles for my Arch Linux + DWM.

### Programs which have configs in this repository

- dwm (window manager)
- neovim (text editor)
- ranger (file manager)
- termite (terminal)
- zsh (shell)

## Installing

To install, just run:

```
git clone https://github.com/fplucas/dotfiles.git
cd dotfiles
./install.sh
```
